Level Two

Restrictions
Only use HTML (no CSS of any kind) In addition to the elements allowed in Level 1, use any of the following HTML elements: div span h1, h2, h3, h4, h5, h6 strong em ol, ul, li a (Do not link to external sites, but you can link to parts of your poem using an attribute to assign an id) Again, you can use & n b s p ; (but without spaces) to equal a visible space.
Goals
Focus on presenting the text in different hierarchies and scales using only HTML.
